# Tugas KNN
Implementasi algoritma K-Nearest Neighbor (KNN) menggunakan Python dan dataset Iris dari sklearn.

## Tools

* Python
* Visual Studio Code
* NumPy
* Matplotlib
* Scikit-learn

Install library:

```bash
pip install numpy matplotlib scikit-learn
```

## Dataset
Dataset yang digunakan adalah Iris Dataset dari sklearn.

## Implementasi
Program dibuat menggunakan algoritma KNN manual dengan perhitungan Euclidean Distance.

Nilai K yang diuji:

* 1
* 3
* 5
* 7
* 10
* 15

Program juga dibandingkan dengan KNN dari sklearn untuk melihat hasil accuracy.
